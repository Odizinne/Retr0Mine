# Features

- Save / load game state
- You cannot hit a mine on first click
- QoL features such as reveal by clicking on connected cells

# Shortcuts

- `Ctrl+Q` Quit
- `Ctrl+N` New game
- `Ctrl+S` Save dialog
- `Ctrl+P` Settings
- `F11`    Fullscreen (G4m3r)
