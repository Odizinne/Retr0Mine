Modern UI minesweeper created for a friend.  
Runs on windows | linux.

![image](.assets/screenshot.png)

# Features

- You can save / load game state
- You cannot hit a mine on first click
- Use pure logic (if my algo is not fxcked up)

# Shortcuts

- `Ctrl+Q` Quit
- `Ctrl+N` New game
- `Ctrl+S` Save dialog
